# 🐘 PHP Archive

이 폴더는 **PHP 학습 및 실습 자료**를 보관하는 공간입니다.  
기본 문법, 주요 기능, 예제 코드, 실습 프로젝트를 단계적으로 정리하며, 학습하면서 새로 알게 된 내용과 트러블슈팅 경험도 함께 기록합니다.
