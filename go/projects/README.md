## 📑 작성 가이드
- 각 프로젝트는 **독립 폴더**로 관리
- 폴더 내에 다음 파일 구성 권장
    - `main.go` (프로그램 진입점)
    - `README.md` (프로젝트 설명 및 실행 방법)
    - 필요 시 `internal`, `pkg` 등 구조 활용
- 프로젝트별 `README.md`에는 다음 정보 포함 추천:
    - 프로젝트 개요
    - 기능 및 실행 방법
    - 학습 포인트
    - 관련 참고 자료
