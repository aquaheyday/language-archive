
# 📜 JavaScript (웹 개발 중심)

이 폴더는 **JavaScript 학습 과정 웹 개발 중심**으로 정리한 자료, 예제 코드를 보관하는 공간입니다.  
기본 문법부터 함수형 프로그래밍, 비동기 처리, 브라우저 동작 이해, 웹 개발에 활용되는 JS의 다양한 개념들을 정리합니다.
