# 🐛 Web Server Down Due to Sudden Traffic Spike

## ⚠️ 문제 상황
- 발생 날짜: 2025-05-22  
- 발생 환경:
  - 호스팅: 가비아 클라우드  
  - OS: Rocky Linux 8  
  - 웹 서버: Apache
  - 애플리케이션: PHP
- 재현 방법  
  1. 짧은 시간 안에 평소보다 2배 이상의 요청이 특정 경로로 집중될 경우  
  2. 웹 서버 메모리 초과 발생  
  3. OOM(Out Of Memory) 후 Apache 강제 종료 → 웹 접속 불가
---

## 🔍 원인 분석
