# 🚀 Python Projects

이 폴더는 Python 학습 과정에서 진행한 **실습 프로젝트**를 보관하는 공간입니다.  
기능별 단위 예제는 `examples/`에서 관리하고, 이곳은 **조금 더 완성도 있는 작은 프로젝트**를 저장합니다.  
각 프로젝트는 독립 실행 가능하며, 프로젝트별 구성/설명은 각각의 `README.md`에서 확인할 수 있습니다.
